Help & Support

How does MoveIt work?

MoveIt uses WebRTC for direct Peer-to-Peer transfers and a Cloud-Hybrid API for large or cross-network files.

Is it secure?

Yes, P2P transfers are encrypted end-to-end. Cloud files expire automatically and use signed URLs.
